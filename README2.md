# Простий калькулятор
Консольний калькулятор для виконання базових арифметичних операцій.  
Навчальний проєкт для вивчення Git, GitHub та Python.

## Опис
Цей проєкт представляє собою простий калькулятор з текстовим інтерфейсом,  
який дозволяє виконувати основні математичні операції в інтерактивному режимі.

## Функціональність
- Додавання двох чисел
- Віднімання двох чисел
- Множення двох чисел
- Ділення двох чисел (з перевіркою на ділення на нуль)
- Піднесення до степеня
- Обчислення остачі від ділення (модуль)

## Вимоги
- Python 3.7 або вище
- Windows / macOS / Linux

## Встановлення
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/ВАШ-НІК/calculator-project.git
   ```
2. Перейдіть у папку:
   ```bash
   cd calculator-project
   ```
3. Запустіть програму:
   ```bash
   python calculator.py
   ```

## Використання
1. Оберіть операцію (+, -, *, /, **, %)
2. Введіть перше число
3. Введіть друге число
4. Отримайте результат

### Приклад роботи
```
=== Простий калькулятор ===
Операції: +, -, *, /, **, %
Для виходу введіть ‘exit’

Введіть операцію: +
Введіть перше число: 15
Введіть друге число: 7
Результат: 22.0

Введіть операцію: **
Введіть перше число: 2
Введіть друге число: 8
Результат: 256.0
```
